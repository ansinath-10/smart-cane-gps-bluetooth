# Intelligent Assistive Cane with Vision-Based Navigation and Sensor Fusion

## 📌 Overview
This project presents a smart assistive cane designed to improve mobility for visually impaired individuals using AI and sensor fusion.

## 🚀 Features
- Obstacle detection using ultrasonic sensor
- Object recognition using ESP32-CAM
- GPS-based location tracking
- Audio feedback via buzzer / Bluetooth
- Ergonomic handle design (CAD)

## 🧠 System Components
- ESP32-CAM
- HC-SR04 Ultrasonic Sensor
- NEO-6M GPS Module
- 18650 Li-ion Battery
- Buzzer & Vibration Motor

## 🛠️ Design
The system is divided into:
- Power section (battery)
- Interface section (shaft connection)
- Logic section (ESP32, camera, GPS)
